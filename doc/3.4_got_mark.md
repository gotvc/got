# `got mark`

The `mark` subcommand manages [Bookmarks](./2.4_Bookmarks.md)

## `got mark list [space_name]`
Lists the branches in a Space, by default the local Space. 

## `got mark inspect <name>`
Lists the marks in the local Space. 

## `got mark as <name>`
Creates a new mark at name, pointing to the same Snapshot as the mark at HEAD.
This operation does not change HEAD.

## `got mark delete <name>`
Deletes the mark with name if it exists.
Does not error if the branch does not exist.

## `got mark save <name>`
Sets the root of the mark at name to a Snapshot parsed from standard input.

## `got mark load <name>`
Prints the Snapshot marked at name to standard output.
